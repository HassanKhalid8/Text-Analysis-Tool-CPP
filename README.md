# Text Analysis Tool (C++)

[cite_start]A robust console-based application developed as a Fundamentals of C++ project at the Ghulam Ishaq Khan Institute of Engineering Sciences and Technology[cite: 1, 4]. [cite_start]This tool performs comprehensive linguistic analysis on user-provided text paragraphs to extract helpful insights[cite: 12, 14].

- [cite_start]**Paragraph Merging:** Combine two distinct text inputs into a single string for unified analysis[cite: 13, 26, 39].
- [cite_start]**Vowel Counting:** Calculate total vowel counts for individual or merged paragraphs[cite: 13, 26, 40].
- [cite_start]**Word Tracking:** Locate both the first and all subsequent occurrences of a specific word or phrase[cite: 13, 26, 41, 42].
- [cite_start]**Visual Histograms:** Generate word-frequency histograms in alphabetical or increasing frequency order using C++ Maps[cite: 13, 26, 43, 44].
- [cite_start]**Text Reversal:** Reverse the characters of every word within a selected paragraph[cite: 13, 26, 45].
